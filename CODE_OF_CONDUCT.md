# Code of Conduct

## Our Standard

This project aims to provide a respectful, safe, and constructive environment for everyone.

Examples of expected behavior:

- Be respectful in discussions and reviews.
- Assume good intent and give actionable feedback.
- Focus on the code, the problem, and the proposed change.
- Accept that different levels of experience and perspectives strengthen the project.

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks.
- Discrimination of any kind.
- Trolling, intimidation, or deliberate disruption.
- Publishing private information without explicit permission.

## Scope

This code of conduct applies to project spaces such as issues, pull requests, discussions, releases, and other public interactions related to the repository.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the project maintainer through GitHub.

## Enforcement

The maintainer may remove, edit, or reject comments, commits, issues, and pull requests that violate this code of conduct, and may temporarily or permanently restrict participation when necessary.
